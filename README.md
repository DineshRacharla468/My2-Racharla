[Exponentiation in Sass](https://stackoverflow.com/questions/9549525/exponentiation-in-sass)

~~~
@function pow($number, $exponent) {
  $value: 1;

  @if $exponent > 0 {
    @for $i from 1 through $exponent {
      $value: $value * $number;
    }
  }

  @return $value;
}
~~~
